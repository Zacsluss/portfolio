## 💡 About This Portfolio

An interactive, GPU-accelerated portfolio showcasing **30,000 particles at 60 FPS** running entirely in the browser. This project demonstrates expertise in:

- ✨ **Advanced WebGL & 3D Graphics** - Custom GLSL shaders with real-time particle physics
- ⚡ **Performance Optimization** - Adaptive rendering for desktop and mobile devices
- 🎨 **Modern UI/UX** - Fluid animations with glassmorphism design principles
- 📱 **Responsive Design** - Progressive enhancement from mobile to desktop

```
React 19 + Three.js + WebGL 2.0 + Custom GLSL Shaders
├─ 30,000 particles @ 60 FPS (desktop)
├─ 10,000 particles @ 30 FPS (mobile, adaptive)
├─ Real-time mouse interaction & particle morphing
├─ Interactive 3D text with dynamic animations
└─ Optimized bundle: < 250kb gzipped
```

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/Zacsluss/portfolio.git
cd portfolio/particle-portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Deploy to GitHub Pages
npm run deploy
```
